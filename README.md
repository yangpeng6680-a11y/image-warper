# 🌸 图片拉伸编辑器

一个纯前端的四角透视变形工具，支持背景合成与批量导出。

## 功能特点

- 🎯 **四角透视变形** — 拖拽控制点自由调整图片形状
- 🖼️ **背景合成** — 可上传背景底图，前景图合成在上面
- 📦 **批量处理** — 多张图片应用相同变形，一键导出
- 📥 **高清导出** — 预览低分辨率，导出保持原图分辨率

## 在线使用

直接在浏览器打开 `index.html` 即可使用，无需服务器。

**访问地址：** https://yangpeng6680-a11y.github.io/image-warper

（如果你已开启 GitHub Pages，访问上面链接即可）

## 技术栈

- 纯原生 HTML/CSS/JavaScript，无任何依赖
- 双线性插值 + Alpha 混合算法实现透视变换
- 支持 PNG/JPEG 图片上传

## 部署说明

如需使用 GitHub Pages：

1. 进入仓库 **Settings → Pages**
2. Source 选择 ** Deploy from a branch**
3. Branch 选择 **main**，文件夹选 **/ (root)**
4. 点击 Save，等待 1-2 分钟即可访问
